# Community Documentation

This directory contains **unofficial** documentation, guides, and wiki articles maintained by the Project Zomboid community.

> [!NOTE]
> These documents are sourced from the community (PZwiki, GitHub guides, Discord) and may contain speculative information or best practices that differ from official recommendations.

## Directory structure

### `42.1/`

Documentation archived or specific to the early **Build 42.1** beta.

- [Random.md](42.1/Random.md) - Guide on random number generation and seeds.

### `42.13_Unstable/`

Community notes and observations for the **Build 42.13 Unstable** branch.

- [B42_13_Migration_Notes.md](42.13_Unstable/B42_13_Migration_Notes.md) - Community-observed API changes.
- [Mod_data.md](42.13_Unstable/Mod_data.md) - Guides on persistent data storage (ModData).

### `Common/`

General resources, tutorials, and guides that differ from the official documentation or provide more implementation details.

- [README.md](Common/README.md)
