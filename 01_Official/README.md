# Official Documentation

This directory contains official documentation sourced directly from **The Indie Stone**, internal tooling, or decompiled artifacts verified against the game's source code.

> [!IMPORTANT]
> These documents are **normative**. In cases of conflict between this directory and the Community Wiki or other external sources, the information here takes precedence for the specific build versions covered.

## Directory Structure

### `42.13_Unstable/`

Documentation specific to the **Build 42.13 Unstable** branch.

- [Migration_Guide.md](42.13_Unstable/Migration_Guide.md) - Critical changes from B41 to B42.
- [Project_Zomboid_API_for_Inventory_Items.md](42.13_Unstable/Project_Zomboid_API_for_Inventory_Items.md) - New registries, Lua/Java API changes, and item system updates.

### `Common/`

Shared resources, version-agnostic official docs, or guides that apply across multiple recent builds.

## Usage

Refer to [Migration_Guide.md](42.13_Unstable/Migration_Guide.md) in the relevant version folder for a high-level overview of breaking changes.
